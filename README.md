# 🚀 Evalora AI  
Meaning, Measured Right.

Evalora AI is a full-stack AI-powered application that evaluates translation quality using advanced models like COMET, 
while comparing outputs from multiple translation engines. Its aimed for German_English translataion accuracy

---

# Features

- 🌐 Google Translation integration  
- 🤖 DeepL API support  
- 📊 COMET-based translation evaluation  
- 💬 Chat-style modern UI  
- ⚡ FastAPI backend + Next.js frontend  

---

# Tech Stack

Frontend
- Next.js (React)
- Tailwind CSS
- Framer Motion

Backend
- FastAPI
- COMET (ML model)
- DeepL API
- Google Translator

#Setup Instructions

## Backend
cd backend  
pip install -r requirements.txt  
uvicorn main:app --reload  

---

## Frontend
cd frontend  
npm install  
npm run dev  

---

# Environment Variables

Create a `.env` file inside the backend folder:
DEEPL_API_KEY=your_api_key_here
